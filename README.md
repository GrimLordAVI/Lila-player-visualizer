# Lila-player-visualizer
# Player Telemetry Visualization Tool

A web-based visualization tool for exploring player behavior on game maps.

## Features

- Player journey visualization
- Bot vs human differentiation
- Event markers (kills, loot, storm deaths)
- Timeline playback
- Heatmap overlays
- Interactive filtering

## Tech Stack

Frontend:
- HTML
- JavaScript
- Canvas

Backend/Data Processing:
- Python
- Parquet → JSON pipeline

## Run Locally

Start server:

python -m http.server 8000

Open:

http://localhost:8000/web
